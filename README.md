# Exam Scheduler

A simple Java console-based exam scheduling system that allows users to add exams, view scheduled exams, and prevent duplicate exam dates.

## Features
- Add exam subjects and dates
- View exam timetable
- Prevent duplicate exam date bookings
- Simple menu-driven system

## Technologies Used
- Java
- ArrayList
- Scanner class
- OOP basics

## How to Run
1. Open the project in a Java IDE
2. Run the `Exam.java` file
3. Use the menu to add or view exams
